## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/DevShivam-maurya/portfolio.git
```

Move into project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
